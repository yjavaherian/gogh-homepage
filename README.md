# Goph rewrite

this is my attempt to rewrite [goph](https://gogh-co.github.io/Gogh/) showcase website using a more modern stack.

## Stack

- Deno
- Vite
- React
- Tailwind
- Lucide Icons

## Development

- clone the repo.
- install deno.
- run `deno task dev`

## Build for production

```bash
$ deno task build
```
